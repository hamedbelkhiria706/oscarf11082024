Sélecteurs CSS

Appliquer le CSS à différentes parties de HTML

1. Le sélecteur d'élément cible les éléments en fonction de leur nom de balise HTML.

  1. Les sélecteurs de classe ciblent les éléments en fonction de la valeur de l'attribut de classe.
  2. Les sélecteurs d'ID ciblent les éléments en fonction de la valeur de l'attribut id.
  3. Les sélecteurs d'attribut ciblent les éléments en fonction de leurs attributs et de leurs valeurs.
  4. Le sélecteur universel cible tous les éléments.
